------------------Contenu du compte rendu----------------

	Ce compte-rendu contient essentiellement :
		Un dossier bin/ , qui contiendra tout les binaires compilés
		Un dossier obj/ , qui contiendra les fichiers objet compilés
		Un dossier src/ , contenant les fichiers sources
		Un dossier include/ , contenant les fichier en-tête C++.
		Un dossier tmp/ , contenant tout. 
		Un fichier Makefile.
		Un fichier RapportCA2.pdf, rapport du projet.
		
------------------How to use ?----------------

	Pour utiliser notre programme, il faut d'abord le compiler via la commande make.
	Une fois le programme compilé, il faut le lancer via la commande :
		
		bin/cpp/main <nom_fichier>
		
		Où nom_fichier correspond au fichier assembleur avec lequel on souhaite faire le test.
		Des fichiers assembleurs sont fournis dans le dossier bin/examples/
